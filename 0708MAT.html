7.8 Matching Questions (MAT)

Matching questions ask the student to match one or more questions with one of a set of given answers. They require a few CSV definitions: the possible answers (items appearing in drop-down boxes), the questions (phrases for which a drop-down box item must be selected by the student), and the definition of the correct question/answer associations.

The fields that can be specified are as follows:

FieldMandatoryDescription
questiontype x Must be the first specified field. It states the type of the question. "MAT", "Matching" or "Matching Question" are all accepted.
title The title of the question. Defaults to "questiontitle".
weighting The score of the question, between 0 and 9999. Defaults to 1.
difficulty The difficulty of the question (1-5). Defaults to 1.
isbonus If this is a bonus question (yes) or not (no). Default no "no".
ismandatory If this is a mandatory question (yes) or not (no). Default no "yes".
text x The text of the question. Must be contained within double quotes, if extending more than one line, or if containing commas. Can include HTML formatting.
image Insert an image at the start of the question. Specifies a single field, which is the image location.
answer x A possible answer to include in drop-down boxes; at least one must be defined. It has two fields: the text that will appear in the drop-down box, and an optional label to identify this answer (defaults to 1), which can be used with the "question" field below.
question x Used to define a question to ask the student; at least one must be defined. It requires two fields: the question text that will be presented to the student, and the label of the correct choice, as defined with an "answer" definition.
score x The scoring approach to use. A single field is expected, with one of the following specifications:
  • "equal": the total points for the question are equally distributed among each of the questions asked;
  • "all": all-or-nothing, i.e. the student must answer all the questions correctly to receive the overall points assigned to this question;
  • right-minus-wrong: same as "equal", but with negative scoring; zero is the minimum score (e.g.if there were four matching options, and one was correct and three incorrect, the total score would be 0 points).
hint A hint shown to the students when answering the question in a quiz. Can include HTML formatting.
feedback Feedback to show to the students, when visualising their test and answers after taking the test (it is not shown during test taking). Can include HTML formatting.
keyword Defines a keyword. Can have an optional second field with a value. These are not uploaded to Brightspace.
keywords Same as before, but several keywords can be defined. No value will be associated with each keyword.
