2. Installation

As a registered developer license has not been acquired yet, both MacOS and Windows installation procedures are somewhat complex.

2.1 MacOS

After downloading the DMG file, double-click it to mount the volume. It contains a PKG file (the installer), and a quick start PDF document.

Double-click the pkg file, and go through a standard MacOS installation procedure; you may need to type the system's password to proceed (this is so that the software can be installed in your Applications folder). Once the installation is complete, you can choose to remove the installer from your machine.

2.2 Windows

Just as with MacOS, installation on a Windows system requires a few steps to bypass security. When downloading the EXE file, browsers such as Microsoft Edge may complain; you will have to confirm your intent to keep the file:

Once you do this, Microsoft Defender will likely trigger a final security check; you have to choose "Show More" and then "Keep Anyway":

Finally, once you double-click the installer file to execute it, Microsoft Defender might still complain; as before, press "More Info" and then "Run Anyway".

Once you are finally able to execute the file, the installation process can begin:

As with the Mac version, the installer goes through a typical Windows installation process; for better results, try to stick to the default choices. Once the installation is finished, you will find a "BSGenerator" folder in your Start Menu, with the executable "BSGenerator" inside it.

2.3 Uninstall

BSGenerator does not provide a specific uninstall, so you can use the standard uninstalling procedure for your operating system:
  • For Windows, go to the main Windows "Settings" application, then choose "Apps", "Apps & features"; then click the triple verticals dots next to BSGenerator, and choose "Uninstall".
  • For MacOS, head over to "Applications", and just drag the BSGenerator icon to the recycle bin.
