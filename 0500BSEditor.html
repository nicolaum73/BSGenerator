5. BSEditor

BSEditor is the graphical-user interface to BSGenerator, responsible for single question creation. In time, it will be integrated into a single BSGenerator interface.

Its main application window application is composed of four main areas:

  • On the left side, the large text editor is the Question Editor, where questions in the BSGenerator specific CSV format are loaded and edited.
  • On the right side, the Question Preview panel is a space reserved for previewing how a question will appear to users in Brightspace.
  • The Log Display panel (bottom right) displays information concerning actions taken when using the application.
  • Above the Question Editor and Question Preview sits the Button Toolbar, which offers shortcut buttons for most of the application's functions.

Further to these panels, a set of menus provides access to all the functions offered by the application.

5.1 Question Editor

This editor allows you to create, load, edit and save questions in CSV format. You can have several files open, which will be organised with tabs. The File, Edit and Question menus offer options related to the content of the currently selected tab.

5.2 Question Preview

This area displays a preview of the question being edited, when that function is activated (either through the menu Question / Preview, or with the Preview button). This is a realistic approximation of the question appearance in Brightspace. If more than one question variation is generated, you can use the navigation buttons to switch between different variations.

It also shows which are the correct answers, the question feedback, and the score the current answer will achieve. This is a dynamic display; try out different question answers, and the score will automatically adjust.

5.3 Log Display

This area keeps track of most operations, along with the date and time at which these were activated (the data is purely local). It is helpful when compiling questions, to keep track of potential problems. It can be exported into a text file, and its contents can be cleared at any time. These functionalities are available through the Tools menu, or with the corresponding shortcut buttons.

5.4 Button Toolbar

The Button Toolbar provides shortcuts for all major functionalities. These buttons implement the following functions (from left to right):
  • create a New (blank) question;
  • Open an existing question;
  • Save the current question;
  • Save a Copy of the current question;
  • Import a question from a QTI archive;
  • Export a question into a QTI archive;
  • Undo the last edit;
  • Redo a previously undone edit;
  • Cut text;
  • Copy text;
  • Paste text;
  • choose the Number of question variations to generate;
  • Preview the question(s);
  • display the generated XML Code;
  • display the generated CSV Code;
  • display the used Python variables;
  • Export the Log from the Log Display;
  • Clear the Log from the Log Display;
  • edit the application's Preferences;
  • open the Help window;
  • Quit the application.
